# renpy-guide

  > Guia completo de **Ren'Py** em PT-BR — crie Visual Novels do zero ao APK.

  Inclui o projeto final **Sakura Café** com sistema de afeição, 3 finais e build para Web e Android.

  ## Online

  - App: https://wallysondevs.github.io/renpy-guide/
  - Repo: https://github.com/Wallysondevs/renpy-guide

  ## Stack

  - React + Vite + TailwindCSS
  - 38 páginas pedagógicas com Terminal, CodeBlock, PracticeBox e AlertBox
  - Tema rosa/roxo Sakura, fonte Quicksand, code highlight via react-syntax-highlighter

  ## Conteúdo

  01. Comece Aqui · 02. Instalação & Setup · 03. Linguagem .rpy · 04. Visual & Cenas
  05. Áudio & Voz · 06. Lógica & Python · 07. Interface (GUI) · 08. Recursos Avançados
  09. Build & Distribuição · 10. **Projeto Final — Sakura Café** · 11. Comunidade & Referências

  ## Dev

  ```bash
  pnpm install
  pnpm dev
  ```
